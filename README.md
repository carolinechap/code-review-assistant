Create a script that takes a PHP file (or code snippet) as input and uses the Mistral API to:
- Detect anti-patterns or bad practices.
- Suggest optimizations (performance, security, readability).
- Explain suggestions as a senior colleague would.

## Requirements
* Php 8.5
* Composer 2 ([download](https://getcomposer.org/download/))
* Mysql 8.0
